CLASS ID STYLE

    Sobre o ID:

  • 1-Forma de identificar um elemento ;
  • 2-Devem ser ÚNICAS para cada elemento
  • 3-Através delas, pode-se atribuir formatação a um elemento em especial.

    Sobre o Style :

  • 1- Usado para diferenciar títulos do texto;
  • 2- Existem diversas cores podendo ser mudadas pelo modelo RGB
  • 3-Através delas pode-se personalizar sua página de acordo a sua vontade
